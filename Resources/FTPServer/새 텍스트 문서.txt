안드로이드 단말기를 ftp 서버로 이용하면 
단말기에 있는 사진, 음악, 동영상 파일등을 PC로 복사할 수 있습니다.

Ftp 클라이언트 프로그램이 이용하거나, 
PC의 파일 탐색기 및 크롬/인터넷 익스플로러 등를 사용할 수도 있습니다.

[ PC에서 사용하기 - 동일한 WIFI 사용 ]
PC와 안드로이드 단말기를 같은 WIFI에 연결합니다. 
안드로이드 단말기에서  FTP 서버를 시작합니다. 
PC에서 파일 탐색기를 이용하여 안드로이드 단말기의 FTP 서버 접속 주소로 접속합니다. 
( 접속 주소는 FTP 서버앱에서 확인 가능합니다. 예: ftp://192.168.123.141:2345 )
사용자 아이디/비밀번호를 묻는 창이 뜨면, 안드로이드 단말기의 FTP 서버에서 설정한 사용자 아이디/비밀번호를 입력하면
연결이 됩니다. 

[필요한 권한]
인터넷 사용 및 저장소 읽기/쓰기 권한이 필요합니다. 

Simple & FTP Server to transfer files to/from your android device with a FTP client ( inc Explorer / Finder / FileZilla).
You can copy your documents, file, photos, musics and files from mobile to your computer (PC).

[How to use]
1. Mobile device and FTP clients are required using the same wireless network.
2. Set your own username, password, port number on FTP Server
3. Start FTP Server
4. Connect to supplied IP using ftp client (inc Explorer / Finder / FileZilla)

[Request Permissions]
- Internet
- Read/Write Storage
