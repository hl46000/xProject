블루투스 연락처 공유앱은 두 대의 안드로이드 폰에서 서로의 연락처를 쉽게 교환 또는 전송을 할 수 있는 앱입니다.

두 대의 단말기를 블루투스로 연결하고 
서로의 연락처를 화면 리스트에서 선택하여 보내거나 받을 수 있습니다. 
내 연락처에서 원하는 연락처를 선택하여 백업 파일을 생성할 수 있습니다. 
생성된 백업 파일은 언제든지 복원할 수 있으며, 
또는 다른 단말기로 보내어 다른 단말기에서도 백업된 연락처를 복원 할 수 있습니다.

[내 단말기]
- 내 단말기의 연락처 정보를 리스트로 보여 줌
- 리스트에서 연락처 클릭 시 연락처의 상세 화면으로 전환
- 연락처를 선택하여 백업하거나 원격단말기로 연락처를 전송할 수 있음
- 연락처를 선택 후( 다중 선택 가능 ) 삭제 할 수 있음


[ 블루투스 연결 ]
- 단말기1과 단말기2 에서 앱 실행
- 단말기1과 단말기2 에서 앱의 우측 하단의 블루투스 버튼을 클릭하여 '블루투스' 화면으로 전환
- 단말기1에서 '다른 장치의 연결 허용' 버튼 클릭
	- 300 초간 연결 허용( '예' 선택 )
- 단말기2에서 '연결 가능한 단말기 찾기' 버튼 클릭
	- 단말기1이 리스트에 보이지 않으면 '연결 가능한 단말기 찾기' 몇 회 반복
	- 단말기1이 리스트에 보이면 단말기1 항목을 리스트에서 클릭
- 단말기1과 단말기2 에서 연결확인 ( 최초 연결 시 )
- 단말기1에서 단말기2의 연결 요청 허용( '예' 선택 )


[원격 단말기]
- 블루트스가 연결이 된 상태
- 우측 하단의 '갱신' 버튼 클릭 
	- 연결된 단말기의 연락처 정보가 리스트로 보여 줌
- 리스트에서 연락처를 선택 후 내 단말기로 가져 올 수 있음


[백업]
- 내 단말기에서 백업한 정보를 리스트로 보여 줌
- 백업 항목을 선택하여 복원 및 삭제 할 수 있음
