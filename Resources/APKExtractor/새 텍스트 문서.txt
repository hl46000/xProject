APK Extractor는 안드로이드 기기에 저장된 앱을 SD카드에 APK형태로 추출합니다.
* 빠르고 사용하기 쉽습니다.
* 시스탬 앱을 포함한 대부분의 앱이 추출 가능합니다.
* 루트 권한을 필요로 하지 않습니다.
* 앱 이름으로 필터링이 가능합니다. 
* 앱 패키지 명으로 필터링이 가능합니다. 
* 추출된 Apk의 기본 저장 위치는 /sdcard/ApkExtractor/ 입니다.


APK Extractor extracts apps stored on Android device to SD card.
* Fast and easy to use.
* Most apps, including system apps, can be extracted.
* Extracts/Share all application.
* No ROOT access required.
* Can filter by app name.
* Can filter by app package name.
* The default save location for extracted Apk is /sdcard/ApkExtractor/
